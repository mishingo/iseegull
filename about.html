iSeegull

iSeegull is a web application that will facilitate the data gathering of avian wildlife for further analysis. We have constructed this web application for Dr. Walters in our Converging Media Capstone Course at the University of Central Florida. Our Web Application serves the purpose as a desktop website and a mobile device site. Each view has different features that can contribute to the overall research.

Desktop View: The main core feature that differentiates the desktop view from the mobile view, is being able to see the points on the map of the captures that people have made while at mosquito lagoon. The birds and their activity appear at the point that they were detected on the map.

Mobile View: In Mobile View you have the accessibility to participate as a citizen scientist, and capture information about the birds in mosquito lagoon.
